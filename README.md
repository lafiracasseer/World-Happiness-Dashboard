# World Happiness Dashboard

An interactive data visualisation and storytelling project exploring the factors 
that influence happiness across 10 countries (2019–2024), using data from the 
World Happiness Report.

## Research Question
> "What factors influence happiness across countries?"

## Tools Used
- R (dplyr, tidyr, ggplot2, readxl, writexl)
- Microsoft Power BI
- Excel

## Key Visualisations
- Stacked Column Chart, Waterfall Chart, Donut Chart
- Area Chart, Funnel Chart, Correlation Matrix

## Data Source
[World Happiness Report](https://www.worldhappiness.report/data-sharing/)
